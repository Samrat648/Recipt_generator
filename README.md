# 🧾 Receipt Generator

A simple Flask web application to generate printable receipts.

## Features

- Generate Receipt
- Auto Total Calculation
- Current Date & Time
- Print Receipt
- Responsive UI

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```

## Tech Stack

- Python
- Flask
- HTML
- CSS
